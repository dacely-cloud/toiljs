# toiljs docs

Reference documentation for toiljs server-runtime APIs.

- [Cookies](./cookies.md): the cookie library, the `Cookie` builder, the
  `Cookies` parser/codec, `CookieMap`, `SecureCookies` (HMAC signing and
  AES-256-GCM encryption), the `base64url` helpers, and the `Request` /
  `Response` integration. All available as ambient globals (no import, like
  `crypto`) and exported from `toiljs/server/runtime`.
